Instructions

Write a Java application that does the following.

  1. For this program, add on to the code for Assignment #8.
  2. Add a (non-static) method to the Person class that adds the String “Dr.” to the beginning of the name of a Person object. The name of this method should be phd(). The return type is void. The method has no parameters. In other words, this method changes the value of the data field for the name.
  3. Add a (non-static) method to the Person class that adds 1 (one) to the age of a Person object. The name of this method should be birthday(). The return type is void. The method has no parameters. In other words, this method changes the value of the data field for the age.
  4. In the main method, print out the Person object (person variable), using the toString() method. Do NOT use the System.out.println() method, instead use the JOptionPane.showMessageDialog() method.
  5. Then call both the phd() method and birthday() methods. Again, output the object using the toString() method, and the JOptionPane.showMessageDialog() method.
  6. Use the second name and age to instantiate another Person object. Use the toString() method, and the JOptionPane.showMessageDialog() method to output the information.
  7. Then call both the phd() method and birthday() methods for the second Person object. Again, output the object using the toString() method, and the JOptionPane.showMessageDialog() method.
  8. Below your LastnameFirstname9 class, create a second class in your LastnameFirstname9.java file.
  9. Unlike your LastnameFirstname9 class, do NOT include the public modifier. In other words, the code for your Person class should look like this:
class Person{
		//data fields for name and age
		//constructor
		//toString() method
		//phd() method
		//birthday() method
	}
  1. See FractionCalculatorInOneFile.java for an example of several classes in one Java file.
  2. Create two data fields. One data field for name and a second data field for age in your Person class.
  3. Write the constructor for your Person class. You should have a String and Integer (or int) parameters, which initialize your two data fields.
  4. Write the toString() method for your Person class. The return value should have the format: “X is Y years old.”
  5. Write the phd() method for your Person class. The return type is void. The method has no parameters.
  6. Write the birthday() method for your Person class. The return type is void. The method has no parameters.
  7. Make sure your code follows the ICS 211 Java Coding Standard, in particular the Java documentation (Javadoc) comments that go above each method.
  8. Write your original comments every 3-5 lines of code.
  9. WARNING: In the edit method, do NOT copy my code or my comments. Use my code as a guide to write your own code.
  10. Make sure your code follows the ICS 211 Java Coding Standard, in particular the Java documentation (Javadoc) comments that go above each method.
