Instructions

  1. Write a Java application that does the following.
  2. Write an edit method for the program GroceryListProgram4.java.
  3. Save the GroceryListProgram4.java as LastnameFirstname4.java.
  4. Add the edit method to the code. You do not have to change the code or comments for GroceryListProgram4.java program; however, the code and comments in the edit method should NOT be copied from any of my example code.
  5. The edit method should prompt the user for the row number of the item to be replaced, name of the new item, and number of new items.
  6. You need to include error checking. For example, if the user enters a row number that does not exist in the list, you need to display an error message.
  7. Write your original comments every 3-5 lines of code.
  8. Make sure your code follows the ICS 211 Java Coding Standard, in particular the Java documentation (Javadoc) comments that go above each method.
  9. WARNING: In the edit method, do NOT copy my code or my comments. Use my code as a guide to write your own code.
