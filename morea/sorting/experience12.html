Instructions

Write a Java application that does the following.

  1. Use the first argument of commandline input as the input file name.
  2. For example, args[0] will be the name of your input file.
  3. The name of your input file should be typed into the Run Arguments text box at the top of jGRASP.
  4. Your main method should be short with only two (2) method calls. You might also need some error checking for commandline input and other possible errors, so you should use if statements and/or try-catch blocks.
  5. Store each word in the input file in an array of Strings.
  6. Write a method to accomplish this.
  7. The parameter of the method is the input file name.
  8. The method should return an array of Strings with all the words in the input file.
  9. You need to make a connection to the input file and read each word.
  10. If you’re not sure how to read one word at a time from an input file, check out the methods of the Scanner class in the Java API.
  11. Print out all the words in the array in ASCII order by using the compareTo() method of class String. The compareTo() method of class String will be called by the sorting methods in the Sorting.java program. You DO NOT need to write or use the compareTo() method in your code. It will be automatically called by sorting methods in the Sorting.java program.
  12. Write a method to accomplish this.
  13. The parameters of the method are the array of Strings and the file name.
  14. Sort the words in the array in ASCII order by choosing the appropriate sorting method in the Sorting.java program. Note that if you try to sort NULL elements, you will get a NullPointerException. To prevent this, sort from 0 to the last word that is inserted into the array. For example: Sorting.quickSort(array, 0, indexOfLastWord);
  15. To do this, you need to save the Sorting.java program in the same folder as your LastnameFirstname12.java program.
  16. You DO NOT need to change any code in the Sorting.java program.
  17. To prevent the automatic output of the Sorting.java program, use this code before you sort: Sorting.display = false;
  18. You DO NOT need to write a separate sorting method in your LastnameFirstname12.java program. The point of this assignment is to learn how to recycle (reuse) code.
  19. Justify your choice of sorting method in the comments at the top of your program
  20. Print the name of the file.
  21. Print all of the words that are stored in the array.
  22. Your output should be similar to the output below.
  23. Write your original comments every 3-5 lines of code.
  24. WARNING: Do NOT copy my code or my comments. Use my code as a guide to write your own code.
  25. Don’t forget to add “JavaDoc” style comments above each method. See ICS 211 Java Coding Standard (Comments: Methods) for details.
