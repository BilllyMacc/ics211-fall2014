Instructions

Write a Java application that does the following.

  1. Implement a Postfix Calculator using a Stack by using the algorithm in the slides. See slides 35 and 36 titled “Postfix Evaluation Algorithm” for the algorithm.
  2. Use the interface StackInterface.java, the class LinkedStack.java, and the class Node.java in your program.
  3. Save the three files in the same folder (directory) as your LastnameFirstname14.java file.
  4. Do not make any changes to these files.
  5. Your program should use command-line input to read from a file. The file name is only argument on the command-line input, so args[0] is the file name. In you main method, read one line of data from the input file, calculate the result, then output to the screen. Repeat this until you reach the end of the file. Each line in the file will contain a postfix expression.
  6. Make sure your program is robust. It should not crash when a user enters invalid input. Instead, it should display an appropriate error message. Use exceptions and/or if-statements to do this.
  7. Your output should be similar to the output below.
  8. Write your original comments every 3-5 lines of code.
  9. WARNING: Do NOT copy my code or my comments. Use my code as a guide to write your own code.
  10. Don’t forget to add “JavaDoc” style comments above each method. See ICS 211 Java Coding Standard (Comments: Methods) for details.
